# 大厂与研究机构岗位映射

## 我的能力主线

我的能力不应被窄化为“生态碳通量研究”。更准确的表达是：

> 多源时空数据融合 + 遥感生态环境理解 + 机器学习建模 + 物理/生态约束解释 + 可复现科研工程。

## 岗位方向 1：数据科学 / 数据分析

### 适配点

- 多源数据清洗和融合。
- 特征工程和模型评估。
- 复杂结果可视化。
- 业务/科学问题转化为数据问题。

### 可讲项目

HLS、ERA5、EC flux、FFP 的多源样本构建，site-level CV 和模型诊断。

### 需要补强

- SQL。
- A/B test 和因果推断基础。
- 数据仓库和指标体系。

## 岗位方向 2：遥感算法 / GIS / 时空数据

### 适配点

- HLS、Landsat、Sentinel、MODIS。
- 栅格/矢量/点位/时间序列处理。
- 空间匹配、尺度转换和代表性误差。
- 地图和空间可视化。

### 可讲项目

Footprint-aware HLS 碳通量建模，强调 EC 源区与 30 m 像元的观测支持对齐。

### 需要补强

- 更系统的 GDAL/rasterio/geopandas 工程经验。
- 云上遥感处理，如 GEE、STAC、Zarr、对象存储。

## 岗位方向 3：AI for Science / 科学智能

### 适配点

- 地球系统数据。
- Transformer 和多模态时空建模。
- 物理约束 observation operator。
- 科学问题驱动的模型设计。

### 可讲项目

FAT：HLS CNN/embedding + meteorology/time embedding + temporal self-attention + footprint observation operator。

### 需要补强

- 深度学习工程。
- 大模型/基础模型。
- 分布式训练和推理。

## 岗位方向 4：数字能源 / 双碳解决方案

### 适配点

- 碳循环、碳汇、MRV。
- 遥感生态监测。
- 碳数据质量和不确定性。
- 政策与产业需求理解。

### 可讲项目

高分辨率生态碳通量估计可作为碳监测/MRV 中的模型组件。

### 需要补强

- 碳核算标准。
- 碳市场机制。
- 企业能源管理和碳资产管理。

## 岗位方向 5：ESG / 气候风险 / 可持续发展数据

### 适配点

- 环境数据体系。
- 遥感与生态指标。
- 气候极端和生态响应。
- 报告表达和不确定性说明。

### 需要补强

- ISSB/IFRS S2、GHG Protocol 等披露框架。
- 企业场景的数据治理。
