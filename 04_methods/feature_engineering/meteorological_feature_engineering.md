# 气象特征工程

## 目标

将原始气象变量转换为能解释碳通量过程的特征，包括即时状态、累积效应、滞后效应和胁迫指标。

## 基础变量

- 短波辐射 / PAR。
- 气温。
- VPD。
- 降水。
- 土壤水分。
- 风速。
- 相对湿度。
- 时间变量：DOY、hour、season。

## 衍生特征

| 特征 | 含义 | 用途 |
|---|---|---|
| 日平均/日最大温度 | 热环境 | RECO 和热胁迫 |
| 白天累积辐射 | 光合能量 | GPP |
| VPD 最大值/均值 | 大气干旱 | 气孔限制和水分胁迫 |
| 前 7/14/30 天降水 | 水分记忆 | 干旱和恢复 |
| 土壤水分异常 | 水分胁迫 | GPP/RECO 共同影响 |
| DOY sin/cos | 季节周期 | 避免日期断点 |
| hour sin/cos | 日内周期 | 半小时建模 |

## 注意事项

1. 滞后窗口不能使用未来数据。
2. 训练/测试划分前后保持特征构建逻辑一致。
3. 对 ERA5 变量要先核对单位。
4. 对半小时模型，日尺度和半小时尺度特征要分清。

## 我的项目应用

FAT 模型中气象和时间 embedding 与 HLS 空间特征共同输入 Transformer。气象特征提供过程驱动，HLS 提供局地空间状态，footprint operator 提供塔尺度观测支持。
