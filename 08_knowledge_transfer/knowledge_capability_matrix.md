# 知识能力矩阵

这个矩阵用于记录当前知识体系中已经形成的模块、证据来源、可迁移能力和需要继续补强的部分。

| 知识模块 | 已有基础 | 可迁移到的问题 | 待补强 |
|---|---|---|---|
| 多源时空数据处理 | HLS、ERA5、EC flux、FFP 同步建模 | 遥感产品验证、生态监测、气候风险分析 | 更系统的大规模数据工程、云端处理和数据版本管理 |
| 碳循环与生态机制 | NEE/GPP/RECO 分量解释、GPP/RECO 边界判断 | 碳汇评估、生态系统功能评估、MRV | 碳核算方法学、生态模型和土壤碳过程 |
| 观测支持与尺度匹配 | Footprint-aware observation operator | 点位-像元匹配、上尺度、代表性误差评估 | 地统计、不确定性传播、空间抽样设计 |
| 机器学习建模 | Transformer、same-backbone comparison、多任务输出 | 科学机器学习、时空预测、模型诊断 | 深度学习工程、可扩展训练、模型部署 |
| 模型验证 | Site-level CV、leakage 控制、稳健性分析 | 独立验证、跨区域泛化、产品可信度评估 | 因果推断、空间 block CV、外推风险评估 |
| 解释性分析 | 变量重要性、PDP、支持组成诊断、机制边界 | 机制诊断、错误归因、结果沟通 | SHAP、反事实分析、因果图 |
| 科学表达 | 论文、图件、审稿回应、技术报告 | 学术写作、技术文档、政策简报 | 面向不同读者的层级化表达 |
| 政策与应用理解 | 双碳、MRV、碳汇、ESG 初步框架 | 碳监测、生态治理、可持续发展数据系统 | 政策原文、标准体系、行业实践案例 |
