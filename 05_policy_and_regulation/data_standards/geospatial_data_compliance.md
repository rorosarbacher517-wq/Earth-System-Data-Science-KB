# 地理空间数据合规与数据安全

## 为什么需要关注

遥感、GIS、站点观测和企业空间数据常涉及地理位置、生态资源、基础设施和土地利用信息。在企业和政府场景中，数据安全、测绘地理信息合规和数据出境是必须考虑的问题。

## 主要风险

| 风险 | 说明 |
|---|---|
| 地理信息敏感性 | 高精度位置、基础设施、资源分布可能敏感 |
| 数据来源许可 | 遥感、站点、再分析、商业数据有不同使用条款 |
| 数据出境 | 跨境传输可能涉及合规审查 |
| 个人/企业数据 | 企业碳排放、资产位置、生产活动可能是商业敏感数据 |
| 模型产品责任 | 碳核算和 ESG 披露需要可解释、可追溯 |

## 对知识库的要求

1. 每个数据源记录来源、许可和引用方式。
2. 原始数据和可公开样例数据分开。
3. 不把敏感站点、企业数据或未授权数据上传公开仓库。
4. 对外展示时使用样例数据、合成数据或公开数据。
5. 论文、代码和报告中明确数据可用性边界。

## 对求职和企业项目的价值

很多候选人只谈算法，不谈数据合规。你如果能在面试中说明空间数据、碳数据和企业数据的合规风险，会显得更成熟，更接近真实业务。
