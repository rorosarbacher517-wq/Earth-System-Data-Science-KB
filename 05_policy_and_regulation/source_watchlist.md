# 政策法规与权威资料观察清单

这个文件用于记录后续需要下载、核验和归档的政策法规、标准和国际资料。正式归档时，应把原始文件放入 `policies/` 或 `reports/`，并写对应 Markdown 笔记。

## 中国双碳与生态治理

| 主题 | 优先来源 | 需要整理的问题 |
|---|---|---|
| 碳达峰、碳中和顶层设计 | 国务院、国家发改委、生态环境部 | 国家目标、时间表、重点行业、自然碳汇角色 |
| 碳市场与碳排放权交易 | 生态环境部、国务院 | 碳市场机制、配额、核查、数据质量 |
| 温室气体核算与 MRV | 生态环境部、市场监管总局、行业标准 | Monitoring、Reporting、Verification 的数据和方法要求 |
| 林草碳汇和生态碳汇 | 国家林草局、自然资源部、生态环境部 | 碳汇核算、生态系统监测、遥感应用 |
| 生态保护和自然资源监测 | 自然资源部、生态环境部 | 生态红线、国土空间监测、生态质量评估 |
| 数据安全和地理信息合规 | 网信办、自然资源部、测绘地理信息标准 | 数据出境、地理信息安全、遥感数据使用边界 |

## 国际气候治理

| 主题 | 优先来源 | 需要整理的问题 |
|---|---|---|
| Paris Agreement | UNFCCC | 全球减排框架、NDC、透明度机制 |
| Global Stocktake | UNFCCC | 全球盘点如何要求更可靠的监测数据 |
| IPCC AR6 | IPCC | 气候变化科学基础、碳预算、陆地碳汇不确定性 |
| GHG Protocol | WRI/WBCSD | 企业碳核算和供应链排放 |
| ISSB/IFRS S2 | IFRS Foundation | 气候相关财务披露 |
