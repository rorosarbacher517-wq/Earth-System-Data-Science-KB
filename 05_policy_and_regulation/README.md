# 05 政策法规层

政策法规层回答：国家、社会和产业为什么需要这个方向？

这个模块用于整理双碳、碳市场、碳汇核算、生态保护、数据安全、地理信息合规、ESG 和国际气候治理相关内容。它不是为了背政策，而是为了理解研究和技术如何进入真实治理和产业需求。

## 子模块

| 子模块 | 内容 |
|---|---|
| `carbon_neutrality/` | 碳达峰、碳中和、国家战略 |
| `carbon_accounting/` | 碳排放核算、碳汇核算、MRV |
| `carbon_market/` | 全国碳市场、碳交易、配额、核查 |
| `ecological_policy/` | 生态保护、自然资源监测、生态红线 |
| `data_standards/` | 地理信息、数据安全、遥感数据标准 |

## 第一轮需要整理的权威资料

1. 中国碳达峰碳中和顶层文件。
2. 全国碳市场和碳排放权交易规则。
3. 温室气体核算与 MRV 相关标准。
4. 生态系统碳汇、林草碳汇、自然资源监测相关政策。
5. IPCC、UNFCCC、Paris Agreement、Global Stocktake 等国际框架。
6. ESG 和可持续披露相关标准。

## 与我的研究的关系

我的研究关注的是陆地生态系统碳通量估计和空间支持匹配。它可连接到政策需求中的：

- 碳汇监测与评估。
- MRV 中的 Monitoring 和 Verification。
- 区域生态系统碳收支分析。
- 碳中和背景下的自然碳汇和生态产品价值实现。
- 企业 ESG 和气候风险披露中的环境数据底座。
