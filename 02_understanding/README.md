# 02 认识层

认识层回答：我如何看待这个领域？

这里不主要存放教材定义，而是存放我对科学问题、技术边界、产业趋势和个人成长方向的判断。它的价值在于形成“自己的理解”，避免知识库变成无差别收藏夹。

## 需要沉淀的认识

1. 遥感碳监测的核心难题不是只有模型精度，还包括尺度、代表性、验证和不确定性。
2. EC 通量塔与遥感像元之间存在 observation-support mismatch，尤其在高分辨率 HLS 场景下更明显。
3. 机器学习模型的提升需要区分模型容量、输入信息、观测算子和验证设计的贡献。
4. 变量重要性不等于生态机制，相关性不等于因果。
5. 国家双碳、MRV、碳市场和 ESG 需求会推动遥感碳监测从科研问题走向治理和产业问题。
6. 企业需要的不只是论文结论，而是可复现流程、可解释指标、可部署工具和风险可控的数据产品。
