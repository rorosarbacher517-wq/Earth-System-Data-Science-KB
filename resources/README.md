# 资料资产

这个目录存放可以长期维护的资料资产。它不是论文 PDF 堆，而是“可追踪、可引用、可扩展”的资料层。

## 子目录

| 目录 | 内容 |
|---|---|
| `references/` | 核心参考文献 BibTeX、文献主题清单 |
| `data_sources/` | 官方数据源、访问入口、许可和使用边界 |
| `policy_sources/` | 政策、标准、国际框架和官方入口 |
| `glossary/` | 核心术语表 |
| `learning_paths/` | 分阶段学习路线 |

## 原则

1. 公开仓库优先存放元数据、链接、引用和笔记。
2. 受版权限制的论文 PDF 不直接上传。
3. 数据源必须记录访问入口、许可和引用要求。
4. 政策资料优先指向官方原文。
5. 每个资料文件都应能回链到理论、方法、应用或项目模块。
