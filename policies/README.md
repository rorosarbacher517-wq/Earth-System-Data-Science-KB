# 政策法规资料库

这里存放政策、法规、标准、指南和官方文件。政策笔记应优先关注“它改变了什么需求”，而不是只摘录条文。

## 主题目录

| 目录 | 内容 |
|---|---|
| `china_carbon_neutrality/` | 中国双碳顶层政策 |
| `carbon_accounting/` | 碳核算、MRV、温室气体清单 |
| `ecological_conservation/` | 生态保护、自然资源、生态碳汇 |
| `data_and_geospatial/` | 数据安全、地理信息、遥感数据合规 |

## 政策笔记模板

1. 文件名称。
2. 发布机构。
3. 发布时间。
4. 文件级别。
5. 核心要求。
6. 与遥感/碳通量/碳监测的关系。
7. 对企业、社会、国家或世界需求的影响。
8. 对我知识体系和项目方向的启发。
