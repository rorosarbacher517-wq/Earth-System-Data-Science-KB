# 论文资料库

论文 PDF 按主题存放，阅读笔记可以放在同目录，也可以在对应知识模块中建立链接。

## 主题目录

| 目录 | 内容 |
|---|---|
| `theory/` | 基础理论、综述、教科书型论文 |
| `remote_sensing/` | HLS、Landsat、Sentinel、MODIS、SIF、植被指数 |
| `meteorology_climate/` | ERA5、气象驱动、气候变化、极端事件 |
| `ecology_carbon_cycle/` | 碳循环、通量塔、生态生产力、碳汇 |
| `spatiotemporal_data/` | 时空统计、尺度匹配、representativeness、footprint |
| `machine_learning/` | 随机森林、XGBoost、Transformer、AI for Earth |
| `policy_and_application/` | 碳监测、MRV、ESG、行业应用 |
